
# VHD util

## Code style

Prefixes/sufixes:

- `e_` - entity/component instance
- `generate_` - generate statement
- `_v` - variable
- `_t` - type
- `g_NAME` - generic
- `c_NAME` - constant
- `i/o_` - input/output ports
- `_n` - active low

- ports : `std_logic` and `std_logic_vector`
- use `downto`
- `entityname.vhd`
- `entityname_tb.vhd`
- `packagename_pkg.vhd`

- 4 spaces
- ascii
- avoid `std_logic_unsigned`, etc.
